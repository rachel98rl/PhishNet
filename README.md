# ECHacksBack

The backend system for our ECHacks project

# Goto [production](https://github.com/jQwotos/ECHacksBack/tree/production) for live code

## Installation

- Install python3
- Install required third-party packages
  - ```pip3 install -r requirements.txt```

## Usage

```python3 main.py```
